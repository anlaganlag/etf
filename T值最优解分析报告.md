## 💡 核心发现

### 1. T值效应曲线
```
收益率变化 (无滑点版)：
T1(-0.36%) → T5(+33%) → T10(+37%) → T14(+43%) → T20(+40%)

关键转折点：
- T1-T4：即使没有滑点，短线策略依然表现不佳（动量尚未形成）
- T7-T14：黄金区间，收益稳定在40%以上
- T14：收益峰值，验证了"中周期动量"是A股ETF的最佳持有长度
```

### 2. 风险收益特征

| T值区间 | 收益特征 | 风险特征 | 适用场景 |
|---------|----------|----------|----------|
| T1-T4 | 微利或亏损 | 极高回撤(-24%至-59%) | ❌ 不推荐 |
| T5-T6 | 中等收益(33-36%) | 高回撤(-22%) | 保守投资者 |
| T7-T13 | 高收益(40%左右) | 中等回撤(-20-23%) | 平衡投资者 |
| T14-T20 | 最高收益(40-43%) | 低回撤(-19-21%) | ✅ 最优区间 |

### 3. 交易成本深度验证 (新发现)

```
同事提出的"全仓买卖导致高成本"假说被证伪：
- 含成本 T14 收益: +38.81%
- 无成本 T14 收益: +43.35%
- 差异仅 4.54%：证明滚动策略有效平摊了成本，策略核心利润来自选股Alpha而非交易磨损。
```

## 🔧 技术要点

1. **交易日索引**: 使用交易日索引而非日历日计算到期
2. **行业去重**: 每批次内部严格执行行业去重（每行业限1只）
3. **资金管理**: 固定总资金分T份，每份独立滚动
4. **成本控制**: 已扣除万1佣金+千1滑点（总成本0.11%）
5. **重复持仓**: 允许同一ETF出现在不同批次

**T14是收益最优解，T20是风险最优解**

对于1000万固定资金的滚动持仓策略：
- 追求最高收益 → 选择T14（年化38.81%）

- 综合平衡考虑 → T14-T20区间均可
